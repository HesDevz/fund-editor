重大项目产业投资基金（国企投资基金）

国有投资基金 · 天蚨基金管理 · 承上启下 · 核心使命：培育产业、做大项目、落地产能

📋 基金架构

  • 总规模：100亿元，首期20亿元，分期实缴
  • 存续期限：8+2年（投6退2，可延长2年）
  • 组织形式：有限合伙（LP）
  • 管理人（GP）：天蚨基金
  • 架构：母基金 + 子基金 + 直投

🏗️ 投资分档（直投）

  • 小型重大项目：5000万–2亿元
    链上配套、关键零部件产能
  • 中型重大项目：2亿–10亿元
    成果规模化量产、企业扩产升级
  • 大型重大项目：10亿–30亿元
    链主企业重大投资、重点招商落地

💰 资金来源

  • 市财政 / 金控集团：核心出资
  • 国家大基金：争取国家集成电路基金等出资
  • 金融机构：银行、保险、信托配资
  • 外部资本：产业资本、上市公司、社会资本

🔗 前后端衔接

  • 前端衔接（一至三级）：与长兴基金建立项目推荐清单、投资进度共享、接续投资优先权
  • 后端协同（五级）：与并购基金建立项目退出通道、产业整合协同
  • 政府协同：与市科技局、工信局、商务局重大项目信息共享和联合招商

性质定位：市场化运作的产业并购基金  |  核心使命：整合产业链、做大做强龙头企业

投资方式：并购贷款 + 股权投资 + 过桥融资   |   重点方向：上市公司并购、产业链垂直整合、跨区域产业整合

—— 实施路径与保障措施 ——

实施路径 重大项目产业投资基金分阶段推进计划

第一阶段 顶层设计（2026 Q2–Q3）

  1. 完成基金设立方案编制（组织形式、出资结构、治理架构、投资策略、退出机制）
  2. 报市政府审批（市财政局、市金融办审核 → 市政府常务会议审议）
  3. 确定GP管理架构，明确天蚨基金与各方权责边界
  4. 完成基金工商注册（有限合伙企业）、税务登记、银行开户
  5. 中基协备案登记（管理人登记 + 产品备案）

第二阶段 资金募集（2026 Q3–Q4）

  1. 落实市财政出资，纳入年度预算
  2. 国有资本核心出资到位，分批实缴
  3. 对接国家大基金（国家集成电路基金、中国政企合作基金等）
  4. 金融机构配资（银行、保险、信托等机构投资者）
  5. 外部资本引入（产业资本、上市公司等战略投资者）

第三阶段 制度建设（2026 Q4）

  1. 建立投资决策制度
  2. 建立风控管理制度
  3. 建立项目管理制度
  4. 建立绩效考核制度
  5. 建立前端基金衔接制度

第四阶段 项目储备与投资运作（2026 Q3起）

  1. 建立光电重大项目库（对接中国光电城、光机所、应化所、重点企业）
  2. 梳理链主企业扩产需求（奥普光电、长光辰芯、长光卫星等）
  3. 对接重点招商项目（头部光电企业落位长春资本金配套）
  4. 首批子基金落地（光电芯片、激光、商业航天、新型半导体赛道）
  5. 首批直投项目启动（≥5000万，优先链主企业 + 重点招商项目）
  6. 定期评估：每年度评估投资进展、返投完成、产业带动效果

保障措施 六大保障体系

● 组织保障

  • 明确基金筹备工作领导小组单位或人选
  • 明确职责分工
  • 定期调度机制

● 资金保障

  • 财政资金优先保障，纳入年度预算
  • 国有资本持续注资，分批实缴避免闲置
  • 专户管理，资金隔离确保安全
  • 追加出资机制，根据进展适时增资

● 制度保障

  • 投决制度：投决会规则
  • 风控制度：出资方式、止损线、关联交易禁止等
  • 信息披露：季度报告，重大事项及时报告
  • 审计监督：年度第三方审计

● 人才保障

  • 组建专业投资团队，内部抽调 + 市场化招聘
  • 专家咨询机制，光电技术、产业经济、基金法律顾问
  • 绩效激励，与业绩挂钩，GP团队参与Carry分配

● 政策保障

  • 市级配套政策：土地、税收、人才专项支持
  • 省级政策对接：争取省级资金配套和政策倾斜
  • 国家级政策窗口：对接国家大基金，争取出资和项目合作

● 协同保障

  • 前端衔接：与长兴基金项目推荐清单、接续投资优先权
  • 后端协同：与并购基金项目退出通道、产业整合协同
  • 政府信息共享：与科技局、工信局、商务局联合招商
  • 光电城对接：项目落地、场地配套、政策配套联动
